# todo-demo-create-with-ai
This is a sample project created using 'Create With AI' widget

To Create the App using "Create with AI" project widget. Open Android Studio and perform the following steps:-
1. New Project -> "Create With AI" -> Enter following prompt
   
   ``Create a To-Do app with categories, a Room DB, and a 'Smart Summary' section using Material 3.``
<img width="640" height="380" alt="image" src="https://github.com/user-attachments/assets/00aabb6b-1da9-4616-ad9a-19f591e52f5b" />

2. Click **Next**.
3. Enter Name, Package Name, project destination, Minimum SDK and other required confirgurarition for the project.
4. Click **Finish** and wait Android Studio to got into agentic loop.

This will create Project Brief Plan, once you'll approve the plan, the the agent will start creating the implementation plan and update the brief at the same time. Check ``.agent\plan.md file`` for the same.

This project was presented in a talk. Read about the entire talk here: https://medium.com/@ruby.verma10/the-agentic-android-building-apps-at-the-speed-of-thought-8deafc81d77a
